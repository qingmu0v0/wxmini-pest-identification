🚀 使用说明
================================================================================

快速启动（5分钟）:
1. 创建数据库: mysql -u root -p < backend/src/main/resources/db/schema.sql
2. 配置后端: 编辑application.yml，设置数据库和API Key
3. 启动后端: cd backend && mvn spring-boot:run
4. 启动Web: cd web && npm install && npm run dev
5. 访问: http://localhost:8081

详细说明请查看 QUICK_START.md
